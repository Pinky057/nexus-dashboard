# 🗺️ Nexus Dashboard Roadmap

This document outlines the current status and future vision for the Nexus Dashboard.

## 🚀 Version Comparison

| Feature | Free Version | Premium (Pro/Elite) |
|---|---|---|
| **Theme** | Dark Mode (Zinc + Indigo) | Light + Dark Mode + Custom |
| **Tech Stack** | Next.js 14 + TS | Next.js 14 + TS |
| **Sidebar** | Responsive (Floating) | Advanced Workspace Switcher |

## 🆓 Free Version (Current Status)

### ✅ Completed "Elite 9.5" Polish
- [x] **Floating Glass Sidebar**: Redesigned as a detached, floating element with high-end glassmorphism.
- [x] **Global "Hover Lift"**: All dashboard cards respond with `y: -5` and soft glows.
- [x] **Elite UI Kit**: Refactored `Button`, `Card`, and `Input` with glassmorphism and premium geometry.
- [x] **Avatar Identity System**: Standardized Squircle avatars across the Dashboard.
- [x] **Transaction Page Elite Style**: Refactored dashboard transactions table to match 9.5 fidelity.
- [x] **Bento Grid Layout**: Transitioned to a modular, asymmetrical grid for the dashboard.
- [x] **Search Command Palette**: Polished visual UI with backdrop blur and premium typography.
- [x] **Responsive Audit**: Fully optimized for mobile and tablet views.

### 🕒 Planned Free Tier Enhancements
- [ ] **Developer Setup Script**: Automated environment preparation (`npm run setup`).
- [ ] **Skeleton UI Expansion**: Add more perceived performance loaders for all data views.
- [ ] **Expanded UI Kit**: Adding more premium-styled form components.
